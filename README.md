# 🚀 MERN Stack Product Management App

A full-stack MERN (MongoDB, Express, React, Node.js) application for managing products.  
Users can create, update, delete, and view products through a modern React frontend connected to a RESTful backend API.

---

## 📌 Features

✅ Create new products  
✅ View all products  
✅ Update product details  
✅ Delete products  
✅ REST API integration  
✅ Responsive UI using Chakra UI  
✅ MongoDB database integration  
✅ Production-ready deployment setup

---
